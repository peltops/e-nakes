# e-nakes

e-nakes is a mobile apps for healthcare providers to managing child immunization.

Download the latest apk here:
https://github.com/peltops/e-nakes/releases/download/v1.1.7/app-debug.apk

# Features
- Immunization schedules and history
- Making appointments with parents
- Quick check-in to clinic
- Immunization record book
